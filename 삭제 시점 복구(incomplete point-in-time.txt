<< 삭제 시점 복구(incomplete point-in-time recovery-PITR)>>

삭제된 사용자가 Oracle 26ai Free에서 RMAN 복구 후 다시 나타나지 않았다면, 이는 사용자가 삭제되기 전의 시점이 아닌 현재 시점/latest 시점으로 데이터베이스가 복구되었기 때문일 가능성이 높습니다. 사용자를 복원하려면 불완전한 시점 복구(PITR)를 수행해야 합니다. 

1. 사용자를 성공적으로 되돌리려면 백업에서 복원하고 DROP user 명령이 실행되기 직전의  타임스탬프로 복구해야 합니다.  

2. 종료 및 데이터베이스 마운트
$ rman target /
RMAN> SHUTDOUT IMPATE;
RMAN> STARTUP MOUNT;

3. 불완전 복구를 수행합니다.

 RMAN> RUN {
  # 특정 시간 형식을 사용하는 예시. 드롭하기 몇 분 전으로 설정합니다.
  SET UNTIL TIME "TO_DATE('2026-05-08 09:30:31', YYYY-MM-DD HH24:MI:SS')"; 

  RESTORE DATABASE;
  RECOVER DATABASE;
  ALTER DATABASE OPEN RESETLOGS;
}
